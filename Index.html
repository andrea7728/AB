Setup

First, let’s load the necessary libraries and data that will allow us to begin our investigation!

## Libraries to include
library(tidyverse)
library(lubridate)

## Load the data
# Replace the path below with the path to where your data lives
data_path <- "https://datajournalism.tech/wp-content/uploads/2019/10/wichita.csv"
stops <- read.csv(data_path)

# Additional data and fixed values we'll be using
population_2016 <- tibble(
  subject_race = c(
    "asian/pacific islander", "black", "hispanic", "other/unknown","white"
  ),
  num_people = c(19294, 42485, 65090, 16686, 245499)
) %>% 
  mutate(subject_race = as.factor(subject_race))

center_lat <- 37.689789
center_lng <- -97.322720

Benchmark test

We saw before that police stopped white drivers more than black drivers. But we need a baseline comparison to determine if the police are biased against one race.

Data Visualization

##Bar Chart

#data prep
barprep <- stops %>% 
  group_by(subject_race) %>% 
  summarize(
    n = n(),
    prop = n / nrow(.)
  )

#make the plot

bar <- ggplot(barprep,
       aes(x=reorder(subject_race,n), y=n))+
  geom_bar(stat="identity", 
           position="identity", 
           fill="red")+
  geom_hline(yintercept = 0) +
  labs(title="Stopped Drivers by Race",
       subtitle = "White American drivers got stopped the most in the city of Wichita Kansas")+
  coord_flip()

options(scipen=10000)

The above chart is biased as it doesn’t account for demographics population. The following chart is more correct. ##Bar Chart2

#data prep
actual <- stops %>% 
  count(subject_race) %>% 
  left_join(
    population_2016,
    by = "subject_race"
  ) %>% 
  mutate(stop_rate = n / num_people)
  

bar2 <- ggplot(actual,
       aes(x=reorder(subject_race,stop_rate), y=stop_rate))+
  geom_bar(stat="identity", 
           position="identity", 
           fill="red")+
  geom_hline(yintercept = 0) +
  labs(title="Stopped Drivers by Race",
       subtitle = "African American drivers got stopped more than Ahite American and Hispanic American in the city of Wichita Kansas,")+
  coord_flip()

options(scipen=10000)
